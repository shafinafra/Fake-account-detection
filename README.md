# 🔍 Fake Profile Detector

A modern web-based application that analyzes social media profile characteristics to identify whether a profile is likely **Real** or **Fake** using heuristic machine learning logic inspired by Random Forest classification.

## 🚀 Features

- 🎯 Detects fake social media profiles
- 📊 Calculates authenticity score
- 📈 Displays profile indicators
- 🎨 Modern responsive UI
- ⚡ Instant analysis with animated results
- 📱 Mobile-friendly design

## 📥 Input Parameters

The application analyzes the following profile features:

- Followers
- Following
- Number of Posts
- Bio Length
- Profile Picture Availability
- Username Length

---

## 🧠 Detection Logic

The detector evaluates multiple profile characteristics such as:

- Followers-to-following ratio
- Number of posts
- Bio completeness
- Profile picture presence
- Username length
- Unusual following behavior

Each factor contributes to a **Fake Score**, which is converted into an **Authenticity Score**.

---

## 📊 Output

The application provides:

- ✅ Real Profile / 🚨 Fake Profile prediction
- Authenticity Score
- Followers count
- Following count
- Followers/Following ratio
- Number of posts
- Bio length
- Profile picture status
